README.TXT

ไฟล์
1. main.py	| โค้ดรันโปรเเกรมหลักโดยใช้ pygame
2. button.py	| โค้ดฟังก์ชั่นแยก แล้วค่อย import เข้า main.py
2. folder/assets| โฟลเดอร์รวมรูปภาพที่ใช้ในการเล่นเกม
3. score.txt	| ไฟล์เก็บคะแนนจากการเล่นเกม

อุปกรณ์ฮาร์ดเเวร์ที่ใช้
1. Laptop	| เก็บไฟล์และโค้ด และใช้เป็นหน้าจอในการเล่นเกม
2. ESP32 #1	| บอร์ดของผู้เล่นคนที่1
      - ประกอบด้วย OLED*1, Switch*2		 
3. ESP32 #2	| บอร์ดของผู้เล่นคนที่2
   - ประกอบด้วย OLED*1, LDR*1

โครงงานนี้เป็นส่วนหนึ่งของวิชาการพัฒนาฮาร์ดแวร์คอมพิวเตอร์เบื้องต้น (01204114)
คณะวิศวกรรมศาสตร์ ภาควิชาคอมพิวเตอร์  มหาวิทยาลัยเกษตรศาสตร์ บางเขน
โดยมีรายชื่อผู้พัฒนาโครงงานดังนี้
1. พิจักษณ์ ลิ้มล้ำเลิศกุล
2. กิรณา รักประกอบกิจ
3. บุณยาพร บุญทับ
4. สรัล สังข์วร
